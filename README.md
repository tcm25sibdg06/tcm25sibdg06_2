# tcm25sibdg06_2
### gestão_de_tarefas

Este trabalho tem como objetivo desenvolver um sistema de informação para gestão de tarefas, denominado Workity, que permita registar, organizar e acompanhar atividades pessoais e profissionais. O sistema utiliza uma base de dados relacional para armazenar e gerir a informação, possibilitando a atualização do estado das tarefas e a sua organização em diferentes espaços de trabalho. Pretende-se, assim, demonstrar a aplicação de conceitos de modelação e gestão de bases de dados na resolução de um problema do quotidiano.
