# tcm25sibdg06_2
### gestão_de_tarefas

Este trabalho tem como objetivo desenvolver um sistema de informação para gestão de tarefas, denominado Workity, que permita registar, organizar e acompanhar atividades pessoais e profissionais. O sistema utiliza uma base de dados relacional para armazenar e gerir a informação, possibilitando o controlo do progresso das tarefas através de multiplos estados e a su organização estruturada em diferentes espaços de trabalhos vinculados a categorias. Pretende-se, assim, demonstrar a aplicação de conceitos de modelação, normalização e extração de dados (através de vistas relacionais) na resolução de um problema do quotidiano.
